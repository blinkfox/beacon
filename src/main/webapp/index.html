Beacon - 轻量级 svg 徽章实时生成服务

这是一个模仿 shields.io 用来实时生成 svg 徽章的 Java Web 微服务。

特性

Beacon 是一个实时生成 svg 徽章的服务，你可以将本服务的徽章方便快捷的嵌入到你的 Markdown 或者 HTML 文件中。

  • 轻量级、简单快速
  • 易于集成，只需将 svg 徽章的链接信息按需修改即可
  • 支持多种样式风格和颜色主题
